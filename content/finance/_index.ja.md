# 経済データ

経済指標とは、経済活動の状態を示す統計的な数値のことである
{{% /quote %}}

Python でデータを取得し、ロジックを検証し、見栄えの良いグラフへ落とし込む——その一連の流れを実践的なサンプルでまとめています。ローリングβや配当再投資シミュレーションなど、追加したレシピも活用して自社や顧客の経済データ分析に生かしてください。
